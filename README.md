# Portfólio Procópio

Site pessoal e portfólio de Procópio (Designer Gráfico & UX), com apresentação, projetos e contato.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Estrutura
- `src/pages`: páginas
- `src/components`: componentes (atom/molecule/organism)
- `src/data`: dados do site e projetos

## Deploy
Domínio principal: `https://procoopio.com.br`
